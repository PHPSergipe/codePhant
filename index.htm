Realizaçao PHP Sergipe / @phpse

Sobre o PHP

O php é uma linguagem de programação construida na web, para web. Sendo utilizada em larga escala e abrangendo mais de 70% dos sites na internet.

O PHPSE é o grupo de usuários da linguagem PHP oficial do estado de Sergipe. Reconhecido pela Zend - The php company e PHP Brasil Comunidades.
www.phpse.net

CodePhant

O CodePhant é uma iniciativa do grupo de usuários PHP de Sergipe, que tem como objetivo unir e encorajar desenvolvedores da linguagem php no estado.

O queremos?

  1. Pretendemos ser referência em evento PHP no estado.
  2. Divulgar e estimular o uso do php profisionalmente.
  3. Criar um vinculo entre empresas e desenvolvedores.
  4. Estimular o debate e a formação de conhecimento técnico
  5. Criar um bom ambiente para a troca de experiência e conhecimento e expandir a rede de negócios dos participantes
